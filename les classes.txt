                                            P.O.O en JavaScript


Une classe contient les éléments suivants :

    1. Attributs
        -Les attributs représentent les propriétés de la classe.

    2. Fonctions (méthodes) et constructeurs
        -Il existe 3 types de constructeurs :
            Par défaut : initialise les attributs de la classe en mémoire
            int : 0
            float : 0.0 ou 0
            string : null


        -Paramétré : attribue des valeurs aux attributs de la classe
        -Par copie : copie un objet dans un autre objet

    3. Déclaration d’une classe

    Syntaxe :

    class Nom_de_la_classe {
        at1;
        at2;

        constructor(par1, par2) {
            this.at1 = par1;
            this.at2 = par2;
        }
    }

    4. Exemple

    Classe : Étudiant
        Attributs :

          id
          nom
          prénom
          âge
          
        Méthodes :

         saisir()
         afficher()
