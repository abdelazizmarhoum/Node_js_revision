                                                RÉVISION NODE.JS



les étapes pour initialiser un projet Node.js

1 : créer un dossier via : mkdir nom_du_dossier
2 : se placer dans le dossier via : cd nom_du_dossier
3 : installer le package json via : npm init -y      
        => package.json est créé

4 : installer un package via : npm install nom_du_package
        => prompt-sync
            -> c'est pour prendre des entrées utilisateur
            -> exemple : const prompt = require('prompt-sync')();
            -> réponse : const name = prompt('What is your name? ');


        => express
            -> c'est le framework web utilisé
            -> exemple : const express = require('express');
            -> réponse : const app = express();
        
        => fs
            -> c'est pour gérer les fichiers
            -> exemple : const fs = require('fs');

5 : créer un fichier index.js via : touch index.js

6 : lancer le serveur via : node index.js



7 : des autre commandes utiles
    - npm install -g nom_du_package : installe le package globalement
    - npm list : liste tous les packages installés
    - npm uninstall nom_du_package : supprime le package de node_modules et de package.json
    - npm update nom_du_package : met à jour le package
    - npm install : installe tous les packages listés dans package.json
    - npm audit : vérifie les vulnérabilités des packages installés

9 : les boucles (for , while)

    for (let i = 0; i < 10; i++) {
        console.log(i);
    }

    // i c'est une variable qui est utilisée pour compter le nombre de fois que la boucle s'exécute
    // vous pouvez l'appeler comme vous voulez
    // elle commence à 0 et s'incrémente de 1 à chaque tour de boucle
    // la boucle s'arrête quand i est supérieur ou égal à 10

    // exemple : 

    while (condition) {
        // code à exécuter tant que la condition est vraie
    }

    exemple : 
    let a = 1
    while (a <10){
        console.log(a)
        a++
    }

    // les code seront exécutés jusqu'à ce que la condition devienne fausse
    // ici il vas exécuter 10 fois
    // la boucle while va continuer à s'exécuter jusqu'à ce que a soit supérieur ou égal à 10
    // a++ c'est équivalent à a = a + 1
    // a-- c'est équivalent à a = a - 1
