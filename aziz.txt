Description :
Créer un système de gestion des étudiants en utilisant les classes et l'héritage.

Instructions :
1. Créer une classe Étudiant avec :

   - Propriété supplémentaire : plateforme (par exemple, Zoom, Teams)

        console.log(`ID Étudiant : ${this.id}, Nom : ${this.nom}, Âge : ${this.âge}`);

Exercice 2 : Construction d'un Serveur Web Simple avec Express

Description :
Créer un serveur web simple en utilisant Express qui sert du contenu HTML statique en utilisant le module fs.

Instructions :
1. Configurer un serveur Express qui écoute sur le port 3000
2. Créer un fichier HTML nommé "étudiants.html" avec une liste d'étudiants
3. Implémenter une route GET "/" qui lit et sert le fichier étudiants.html
4. Utiliser fs.readFile() avec la fonction res.end() pour servir le contenu du fichier
5. Gérer les erreurs de manière appropriée si le fichier n'est pas trouvé

Exemple :
const express = require('express');
const fs = require('fs');

const app = express();
const port = 3000;

app.get('/', (req, res) => {
    fs.readFile('./étudiants.html', null, (err, data) => {
        if (err) {
            res.end('Erreur lors du chargement du fichier !');
        } else {
            res.end(data);
        }
    });
});

app.listen(port, () => {
    console.log(`Serveur démarré sur le port ${port}`);
});

Exercice 3 : Système d'Inventaire de Produits

Description :
Créer un système d'inventaire de produits qui démontre la hiérarchie de classes et l'interaction utilisateur.

Instructions :
1. Créer une classe Produit avec :
   - Propriétés : code, nom, quantité, prix
   - Constructeur pour initialiser ces propriétés
   - Méthode info() pour afficher les informations du produit
   - Méthode calculerValeur() qui retourne quantité * prix

2. Créer une classe ProduitÉlectronique qui étend Produit avec :
   - Propriété supplémentaire : périodeGarantie
   - Redéfinir la méthode info() pour inclure les informations de garantie

3. Créer une classe ProduitAlimentaire qui étend Produit avec :
   - Propriété supplémentaire : dateExpiration
   - Redéfinir la méthode info() pour inclure la date d'expiration

4. Créer des instances de chaque type de produit et afficher leurs informations
5. Calculer et afficher la valeur totale de l'inventaire

Exemple :
class Produit {
    constructor(code, nom, quantité, prix) {
        this.code = code;
        this.nom = nom;
        this.quantité = quantité;
        this.prix = prix;
    }

    info() {
        console.log(`Code : ${this.code}, Nom : ${this.nom}, Quantité : ${this.quantité}, Prix : ${this.prix}$`);
    }

    calculerValeur() {
        return this.quantité * this.prix;
    }
}

Exercice 4 : Manipulation de Boucles avec for et while

Description :
Créer un programme qui démontre l'utilisation des boucles for et while pour résoudre des problèmes mathématiques simples.

Instructions :
1. Créer une fonction qui utilise une boucle for pour afficher les nombres de 1 à 10
2. Créer une fonction qui utilise une boucle while pour calculer la somme des nombres de 1 à N (N saisi par l'utilisateur)
3. Créer une fonction qui utilise une boucle for pour afficher la table de multiplication d'un nombre saisi par l'utilisateur
4. Créer une fonction qui utilise une boucle while pour afficher les nombres pairs de 2 à 20

Exemple :
const prompt = require('prompt-sync')();

// Boucle for pour afficher les nombres de 1 à 10
for (let i = 1; i <= 10; i++) {
    console.log(i);
}

// Boucle while pour calculer la somme
let n = parseInt(prompt('Entrez un nombre : '));
let somme = 0;
let compteur = 1;
while (compteur <= n) {
    somme += compteur;
    compteur++;
}
console.log(`La somme des nombres de 1 à ${n} est : ${somme}`);
