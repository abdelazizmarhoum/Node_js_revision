                                EXPRESS EN NODEJS

1 : definition de express
Express est un cadre minimaliste pour construire des applications Web et des API RESTful avec Node.js.
2 : creation d une application express simple
Pour créer une application express simple, vous devez suivre ces étapes :

    a- Installer express :
    npm install express

    b- Importer express :
    const express = require('express');

    c- Créer une application express :
    const app = express();

    d- Définir le port :
    const port = 3000;
        - vous pouvez utiliser le fichier .env qui contain se code la : 
          PORT = 3000
          et voici comment vous pouvez l'utiliser en votre js fichier: 
          const port = process.env.PORT || 3000; -> si process.env.PORT n'existe pas, alors utiliser 3000
        
    e- démarer le serveur (il execute dans le terminal) : 
    app.listen(port, () => {
      console.log("demarer sur le port ", port);
    });

    f- definir des routes (il execute dans le navigateur dont le url http://localhost:3000/) :
    app.get('/', (req, res) => {
      res.send('hello world');
    });

    // req -> request
    // res -> response
    // listen -> demarrer le serveur
    // get -> methode http get
